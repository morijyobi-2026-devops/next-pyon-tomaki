修正版：個人用アプリの基本設計

複数学期対応 + ログイン機能

アプリ構成（修正）

📱 個人用アプリの構成

🔧 初回セットアップ

  1. ログイン（認証）
  2. 前期の時間割を入力
    • 授業名・曜日・時限・教室
  3. Slack Webhook を設定
  4. 後期の時間割も同様に入力

📱 日常使用

  • ログイン
  • 学期を選択（前期 / 後期）
  • ホーム画面表示
    • 次の授業を大きく表示
    • 授業終了5分前に Slack 通知

📊 データ構造

{
  user: {
    id: "user_001",
    password: "hashed_password",
    slack_webhook_url: "https://hooks.slack.com/..."
  },
  schedules: [
    {
      id: "schedule_spring",
      term: "前期",
      classes: [
        {
          name: "線形代数",
          day: "Monday",
          period: 1,
          start_time: "08:50",
          end_time: "10:20",
          room: "101教室"
        }
      ]
    },
    {
      id: "schedule_fall",
      term: "後期",
      classes: [ ... ]
    }
  ]
}
